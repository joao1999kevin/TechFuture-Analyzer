# TechFuture Analyzer 2.0

## Proyecto Integrador
El impacto de las nuevas tecnologías en la sociedad: visualización del futuro

---

## Descripción

TechFuture Analyzer 2.0 es un software desarrollado en Python que permite analizar el impacto de tecnologías emergentes en la sociedad.

El sistema evalúa tres dimensiones:

- Impacto Productivo
- Impacto Social
- Impacto Ético

Generando un índice promedio de transformación tecnológica.

---

## Tecnologías Analizadas

- Inteligencia Artificial
- Big Data
- Internet de las Cosas
- Realidad Virtual

---

## Arquitectura del Sistema

El sistema está dividido en:

- Capa de Datos
- Capa Lógica
- Capa de Presentación

Aplicando principios de programación funcional y modularidad.

---

## Funcionalidades

- Selección de tecnología
- Evaluación por puntuación (1–10)
- Cálculo de índice promedio
- Clasificación del nivel de impacto
- Visualización estructurada de resultados
